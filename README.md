### **دستیار صوتی فارسی با الهام از J.A.R.V.I.S (Iron Man)**

دستیار صوتی فارسی با قابلیت کنترل سیستم، اجرای دستورات، پاسخ صوتی، ربات تلگرام، مدیریت فایل‌ها، ارسال ایمیل، پخش موسیقی و بسیاری امکانات دیگر.

# 🚀 نصب و راه‌اندازی

## 1️⃣ نصب پیش‌نیازها

```bash
pip install -r requirements.txt
```

---

## 2️⃣ ساخت فایل تنظیمات

داخل پوشه پروژه یک فایل با نام `.env` ایجاد کنید.

سپس اطلاعات زیر را داخل آن قرار دهید:

```env
TELEGRAM_TOKEN=your_telegram_token
YOUR_CHAT_ID=your_chat_id

EMAIL_ADDRESS=your_email@gmail.com
EMAIL_PASSWORD=your_app_password

WOLFRAM_APP_ID=your_wolfram_id
WEATHER_API_KEY=your_weather_key
```

# 🔒 امنیت

فایل `.env` شامل اطلاعات حساس مانند:

- توکن ربات تلگرام
- رمز برنامه‌ای Gmail
- کلیدهای API

است و **نباید** در GitHub قرار بگیرد.

به همین دلیل این فایل داخل `.gitignore` قرار گرفته است.

برای اجرای پروژه:

1. فایل `.env` را ایجاد کنید.
2. اطلاعات خود را طبق `.env.example` وارد کنید.
